# **「我有事 / 我有空」- 基于 LBS \+ AI 的本地互助撮合平台 (MVP版)**

## **一、 产品愿景与定位**

* **产品形态：** 移动端响应式 Web 网页（H5），后续可无缝拓展为 PWA 或小程序。  
* **核心定位：** 一个由 AI 驱动底层意图解析、基于地图（LBS）展现的本地“计划性”任务撮合平台。  
* **商业策略：** 早期去中心化，完全免费，平台不介入交易资金流，仅作为信息高效集散与信任过滤的基建。

## **二、 核心角色定义**

1. **“我有事”（发单方/需求侧）：** 需要在特定时间、特定地点寻求他人帮助（生活琐事或专业技能）的用户。  
2. **“我有空”（接单方/供给侧）：** 拥有空闲时间或特定技能，希望在周边寻找兼职机会或提供互助的用户。

## **三、 核心功能模块与用户体验流**

### **1\. 流量引入与安全层 (增长与风控)**

* **无感人机验证：** 采用 Cloudflare Turnstile 或同等无感方案拦截爬虫，保障真人访问流畅度。  
* **“迷雾地图”游客模式（Growth Hook）：** \* 未登录游客打开网页，直接展示全屏地图。  
  * 地图上仅显示模糊化处理的“需求气泡”（如：\[距离 500米\] 某任务 · ￥50）。  
  * 游客点击气泡查看详情时，弹出极简登录框（手机号验证码/微信授权），利用好奇心与利益点驱动注册。

### **2\. “我有事”：极简发单系统**

* **交互入口：** 首页底部常驻巨大的“文本输入框”（类似聊天框）。  
* **AI 意图解析核心（降本增效）：**  
  * 用户输入自然语言（例：“下周二下午3点找人搬沙发到楼下，50块”）。  
  * 后台调用轻量级 LLM，瞬间提取并生成结构化 JSON 数据（Tag）：  
    * 时间：计划性时间戳（拦截实时性任务，缓解冷启动匹配压力）。  
    * 位置：结合文本描述与浏览器授权 GPS 自动落位。  
    * 定价模式：一口价 (fixed \= 50\) 或 面议 (bidding \= null)。  
    * 任务类型：区分“基础跑腿”与“专业技能”。  
* **发布确认：** 弹出解析后的结构化卡片，用户点击“确认”后，气泡即刻飘至地图上。

### **3\. “我有空”：智能分发与筛选系统**

* **双层任务池与渐进式解锁：**  
  * **新用户（默认状态）：** 地图上仅显示“基础任务”（代拿、排队、搬运等零门槛任务），确保落地有活接。  
  * **技能解锁：** 通过极简的 AI 引导式对话（“平时会点什么手艺？”）为用户默默打上技能 Tag。  
  * **精准定向：** 拥有特定 Tag（如 \[水电维修\]）的用户，地图上才会显现对应的“专业任务”气泡，实现技能与任务的物理隔离。  
* **混合价格筛选器：**  
  * 提供预期收益滑动条（如：100-300元）。  
  * 数据库级筛选：自动展示符合区间的一口价（fixed）任务，**并强制包含**所有待报价（bidding）任务，最大化商机曝光。

### **4\. 沟通与履约闭环 (极轻量化)**

* **BaaS 留言板：** 摒弃沉重的自研 IM 系统，采用云数据库（如 Supabase）搭建轻量级私信面板。  
* **去中心化建联：** 允许并鼓励双方在私信中交换微信号/手机号，将复杂的议价、转账、履约过程转移至微信生态，免除平台前期合规与客服压力。  
* **状态回收机制：** \* “我有事”界面顶部常驻【已招到人 / 关闭任务】按钮。  
  * 配合轻度的平台积分奖励，引导发单方在微信成交后回平台核销任务，清理地图“僵尸气泡”。

## **四、 技术架构建议（低成本起步）**

* **前端：** React / Vue.js 配合移动端 UI 库，构建单页面应用（SPA）全屏地图交互。  
* **后端与数据库：** Serverless 架构（如 Vercel \+ Supabase / Firebase），搞定用户鉴权与轻量级实时数据同步。  
* **地图服务：** 高德 / 腾讯地图 Web API（前期完全免费）。  
* **AI 算力：** 选用高性价比、高响应速度的模型 API（专门做 NLP 信息抽取，成本极低，单次任务仅调用一次）。

## **五、 冷启动行动指南**

1. **切勿盲目买量：** 严控初始地理范围（选定单一大学城或密集型大型居住社区）。  
2. **人工充当“齿轮”：** 早期可通过社群收集需求，创始人手动在后台录入初始任务，确保第一批“我有空”用户打开地图时有内容可看。  
3. **主打“计划性”：** 宣传时重点强调“提前预约”、“周期性兼职”，为平台的撮合效率争取时间窗口。
