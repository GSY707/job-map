# 我有事 / 我有空 · 本地互助撮合平台 MVP Demo

> 一个由 AI 驱动意图解析、基于地图（LBS）展示的本地"计划性"任务撮合平台 H5 Demo

## ✨ 功能特性

### 🌫️ 迷雾地图（游客模式）
- 未登录时地图显示模糊化处理的"需求气泡"（如：`[~500m] ¥**`）
- 点击气泡或登录按钮，触发极简登录弹窗
- 利用好奇心与利益点驱动注册

### 📝 极简发单系统（我有事）
- 首页底部常驻大型输入框，支持自然语言输入
- **AI 意图解析**：输入如"下周二下午3点找人搬沙发，50块"，自动提取：
  - 时间（计划性时间戳）
  - 价格（一口价 / 面议）
  - 任务类型（基础任务 / 专业技能）
  - 标签
- 解析完成后弹出结构化卡片供用户确认
- 发布后气泡即刻飘至地图上

### 🔍 智能分发系统（我有空）
- **类型筛选**：全部 / 基础任务 / 专业技能
- **价格滑动条**：按期望收益区间筛选，自动包含所有面议任务
- **技能解锁**：AI 引导式对话为用户打技能标签，解锁对应专业任务

### 💬 BaaS 留言板
- 轻量级私信面板
- 去中心化建联提示（引导交换微信号）
- 发单方可标记"已招到人"关闭任务，获得平台积分

### 👤 个人中心
- 技能标签管理
- 发布任务历史
- 积分系统

## 🚀 快速启动

```bash
npm install
npm run dev
```

浏览器打开 http://localhost:5173

## 🛠️ 技术栈

| 层次 | 技术选型 |
|------|---------|
| 前端框架 | React 19 + Vite 8 |
| 样式 | Tailwind CSS v4 |
| 地图 | Leaflet.js + react-leaflet |
| 状态管理 | React Context API |
| AI 解析 | 规则引擎模拟（可替换真实 LLM API） |
| 图标 | Emoji（零依赖） |

## 📁 项目结构

```
src/
├── components/
│   ├── MapView.jsx          # 全屏地图 + 任务气泡标记
│   ├── FilterBar.jsx        # 顶部筛选栏（类型 + 价格）
│   ├── BottomBar.jsx        # 底部导航 + 发单按钮
│   ├── GuestBanner.jsx      # 游客迷雾地图提示
│   ├── LoginModal.jsx       # 登录弹窗（手机号 + 微信）
│   ├── PostTaskForm.jsx     # 发单表单（AI 解析流程）
│   ├── SkillOnboard.jsx     # 技能引导对话
│   ├── TaskDetail.jsx       # 任务详情 + 留言板
│   ├── TaskListPanel.jsx    # 任务列表面板
│   ├── ProfilePanel.jsx     # 个人中心
│   └── Notification.jsx     # 全局通知 Toast
├── data/
│   └── mockData.js          # 模拟数据 + AI 解析逻辑
├── store/
│   └── AppContext.jsx       # 全局状态管理
└── index.css                # 全局样式 + Tailwind + Leaflet
```
